# ADAutoPwn — Python dependencies
# Recommended install: pipx (isolated). Pip fallback: pip3 install --user -r requirements.txt
#
# NOTE: ADAutoPwn is a Bash orchestrator. These are the Python-based tools it
# drives. System packages (nmap, smbclient, smbmap, rpcclient, ldap-utils,
# ntpdate, enum4linux-ng, john, hashcat, seclists) and kerbrute (Go binary in
# /opt/kerbrute) are handled by install.sh, not by pip.
#
# OPTIONAL — ADAutoGraph, the local BloodHound-style web UI that ADAutoPwn
# auto-launches at the end of a run (pure Python stdlib, no pip deps). install.sh
# clones it next to this repo; or grab it yourself:
#     git clone https://github.com/C4sh3R/ADAutoGraph
# (override its location with ADAUTOGRAPH_DIR, disable the launch with --no-web)

netexec            # nxc — core SMB/LDAP/WinRM swiss-army knife
impacket           # secretsdump, GetUserSPNs, GetNPUsers, getTGT, dacledit, etc.
certipy-ad         # ADCS enumeration & ESC abuse
bloodhound         # bloodhound-python ingestor
bloodyAD           # fast LDAP privilege/ACL abuse
ldapdomaindump     # offline LDAP dump (optional)
msoffcrypto-tool   # decrypt password-protected Office docs found on shares
volatility3        # hashdump/lsadump/cachedump of *.vmem/*.dmp memory images looted from shares
